# Banjar Adat Sental Kawan Web App 🌴

A modern, digital administration platform built for Banjar Adat Sental Kawan in Desa Adat Ped, Nusa Penida. This project bridges traditional Balinese community heritage with modern web technology, featuring a sleek Glassmorphism UI and native Balinese typography.

## 🎯 Project Vision
To digitalize and streamline the administrative duties of the Banjar Adat. This platform serves as a central hub for the *krama* (community members) to access transparent financial records, view upcoming ceremonial events, and understand the leadership structure, while providing the *Prajuru* (officials) with a secure management dashboard.

## 🚀 Tech Stack
* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS (Focus on `backdrop-blur` for Glassmorphism)
* **Typography:** Inter (Latin) & Noto Sans Balinese (Traditional Script)
* **Database & Auth:** Supabase (PostgreSQL)
* **Hosting:** Vercel

## ✨ Core Features
* **Prajuru & Krama Directory:** A visual hierarchy of the Banjar's leadership and a searchable database of all 128 registered members.
* **Financial Ledger:** A transparent, read-only summary of the Banjar's community income and expenses.
* **Notice Board:** Real-time digital announcements for *Piodalan*, *Ngaben*, and official village regulations.
* **Secure Admin Dashboard:** Protected routes for authorized officials to perform CRUD operations on members, events, and financial records.
